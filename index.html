部署說明
如何部署到 GitHub Pages
1. 在 GitHub 建立新 Repository
2. 上傳全部檔案（index.html、sw.js、manifest.json、icons/）
3. 進入 Settings → Pages → 選 main branch
4. 幾分鐘後網址即可使用
5. 用手機瀏覽器開啟網址
6. 點「加入主畫面」安裝為 App
7. 開啟通知授權，完成！
